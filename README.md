# 智能空间站（Smart Space Station）

## 项目背景

本项目为网页设计大赛参赛作品，主题为“智能空间站”。  
项目以未来空间站的应用场景为设想，探索在现有 Web 技术条件下，
如何通过浏览器端实现具有沉浸感与交互性的展示页面。

## 项目简介

智能空间站是一个基于 HTML / CSS / JavaScript 的前端演示项目，
通过多媒体内容（视频、图片）与模块化页面结构，
构建一个面向未来空间站的可视化交互界面。

项目重点不在于复杂业务逻辑，而在于：

- 页面结构设计
- 场景组织方式
- 多媒体与交互体验的结合

## 设计目标

- 构建清晰的多模块页面结构，便于扩展不同功能场景  
- 利用视频与视觉元素增强沉浸感  
- 在纯前端条件下实现较完整的交互展示流程  

## 主要功能

- 场景模块展示与切换
- 视频与图片等多媒体内容播放
- 模块化资源组织，支持后续功能扩展
- 浏览器端直接运行，无需后端依赖

## 运行方式

项目为纯前端项目，可直接在浏览器中运行。

### 方式一：直接打开

- 双击 `index.html`
- 或通过浏览器“打开文件”
