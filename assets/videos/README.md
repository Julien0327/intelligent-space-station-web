# Video Assets Notice

This project was originally developed for a web design competition
and includes high-resolution video assets for offline demonstration.

Due to GitHub file size limits and GitHub Pages constraints,
video files are intentionally excluded from this repository.

## Local Demo Files (Not Included)

The original demo uses the following local video files:

- background.mp4
- explore.mp4
- china-station.mp4

These files are only required for offline or competition presentation.

## Online Preview

The GitHub Pages version focuses on structure, interaction,
and visual layout. Local video playback is omitted or replaced
with fallback images in the online demo.

This design ensures repository cleanliness and deployment stability.
